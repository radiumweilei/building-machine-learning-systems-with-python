
* 安装libpng
brew install libpng

* 安装webp
brew install webp

* 安装tiff
wget http://download.osgeo.org/libtiff/tiff-4.0.9.tar.gz
tar xfz tiff-4.0.9.tar.gz
cd tiff-4.0.9
./configure
make
make install
